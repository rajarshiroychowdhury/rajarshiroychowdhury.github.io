The following lists some of the open source power system tools that might be helpful for students and academic researchers.
  1. Power System Dynamics, DAE modeling, simulation, and control: ANDES
  2. Steady state power flow simulation and optimization for MATLAB and Octave: MATPOWER
  3. Power Distribution System Simulator: OpenDSS
  4. Steady State Power System Analysis: Pandapower
Books and Courses that I recommend for anyone willing to learn power systems (in no specific order) :
  1. Basic Power System Analysis : Stevenson
  2. Dr. Tom Overbye's teaching webpage has made all its courses free to public. It's a treasure trove of knowledge and I highly recommend budding power system engineers to learn from this excellent resource.
